💡 Ideias Fora da Caixa (Insights Inusitados)

Observações que quebram expectativas e abrem oportunidades

💬 Capsicum usa Compliasset como "repositório", não como "ferramenta"

Fluxo: Neo → Análise → Compliasset (upload resultado) → Outro sistema (compliance final). Oportunidade: Reposicionar Compliasset como hub de integração/automação, não como engine de análise.

💬 Algarve prefere IA não porque IA é melhor em compliance, mas porque atualiza mais rápido

O problema não é o produto, é a VELOCIDADE. Oportunidade: Criar "Centro de Inteligência Regulatória" com atualização contínua, feeds de normas, alertas automáticos (tipo CNN para compliance).

💬 Clientes acessam Compliasset apenas para cumprir obrigações mínimas, não para otimizar

Padrão: Capsicum usa só cadastro, Algarve só Aceite Events, Hike uso mínimo. Oportunidade: Criar "Dashboards de Impacto" que mostram economia de tempo, redução de risco, conformidade atingida (value communication).

💬 API do Compliasset é "muito básica" - clientes querem integração de dados, não criação de eventos

Capsicum pede: CPF/CNPJ → API Neo → Classificação automática → Registrar risco. Oportunidade: Redesenhar API como "data ingestion layer" (webhook, eventos, stream) para automatização end-to-end, não apenas criação de tarefas.

💬 Clientes pequenos (4-8 pessoas) são especialistas multifuncionais, sobrecarregados

Nivi: Alexandre = Compliance + Riscos + LGPD + PLD + KYC sozinho. AWR: Ralph + 4 pessoas. Oportunidade: Templates de workflows pre-configurados por perfil (ex: "Compliance Mini" para <5 pessoas), não genéricos.

💬 Pré-operacionais (Nivi) não abandonam, só aguardam o "trigger" certo

Nivi: CVM habilitada, mas 0 fundos operando. Oportunidade: Criar programa "Crescimento em Operação" que ativa features conforme empresa escala (módulo FIDC quando tiver fundos, etc).

💬 Competidores diretos: Neo (análise), IA generativa (regulatórios), Excel (simplicidade)

Não é rival convencional, é alternativa de categoria. Oportunidade: Posicionar como "orquestrador de compliance" (Neo + IA + compliance em um lugar), não como análise standalone.
