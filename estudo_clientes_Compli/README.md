# Estudo com clientes - EvertecReg

Esta pasta e reservada para guardar materiais de pesquisa com clientes, sem misturar com a documentacao principal do projeto.

## Estrutura

- `entrevistas/`: registros resumidos por cliente ou contato.
- `transcricoes/`: transcricoes completas ou parciais de reunioes.
- `sinteses/`: consolidacoes, padroes recorrentes e aprendizados.

## Convencao sugerida

- Um arquivo por reuniao ou entrevista.
- Padrao de nome: `AAAA-MM-DD_cliente_tema.md`
- Se houver anonimização, usar identificadores como `cliente_01`, `cliente_02`.

## Campos uteis

- Data
- Cliente ou identificador
- Participantes
- Contexto da conversa
- Principais dores
- Necessidades citadas
- Riscos ou bloqueios
- Oportunidades percebidas
- Proximos passos
