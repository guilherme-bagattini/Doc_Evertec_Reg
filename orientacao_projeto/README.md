# Orientacao do projeto

Pasta de orientacao para consulta rapida do EvertecReg, com contexto de negocio, resumo tecnico, decisoes de MVP e pendencias.

## Objetivo

Facilitar o entendimento do projeto para alinhamentos de produto, operacao e tecnologia.

## Como usar esta pasta

1. Comece pelo [Índice rápido de consulta](indice_rapido_evertecreg.md).
2. Para reunioes, use [Guia de reuniao para PO](guia_reuniao_po_evertecreg.md).
3. Para decisoes da etapa atual, veja [Decisões fechadas para o MVP](decisoes_mvp_evertecreg.md).
4. Para pendencias, acompanhe [Dúvidas e decisões em aberto](duvidas_em_aberto_evertecreg.md).
